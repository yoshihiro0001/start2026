# Thinking Constitution

Purpose: MrBrain / Evolution Factoryが判断、実装、案内を行う前に必ず通る思考Routeを定義する。

## Core Hypothesis

Purpose
↓
Thinking Constitution
↓
Trust Rule
↓
Architecture
↓
Workflow
↓
Execution
↓
Evidence
↓
Audit
↓
Learning

## Thinking Route

全ての判断は原則このRouteを通る。

```text
Human Intent
↓
Purpose Check
↓
Human State / Human Cost / Human Risk Check
↓
Goal Validation
↓
Reality / Current Screen / Current Context
↓
Official Docs / Evidence Check
↓
Capability / AI Capability / Implementation Discovery
↓
Options
↓
Decision
↓
Approval
↓
Execution
↓
Result
↓
Learning
↓
Knowledge Compression
↓
Rule / Registry / Gauge Update
```

## Why This Exists

Humanを遅くするためではない。

- 思いつき判断を防ぐ
- AI能力の思い込みを防ぐ
- 古いDocsや料金情報で決めない
- Secret / Recovery / Security事故を防ぐ
- Humanが迷った場所をEvidenceへ変える
- 次回のHuman CostとAI Costを下げる

## Workflow Usage

- Human Navigator: Current ScreenからNext Buttonまで案内する時
- AI Routing: どのAI / toolを使うか決める時
- Security Setup: Recovery / Secret / 2FA / Authを扱う時
- Implementation Discovery: API / Browser / MCP / Human Agentを比較する時
- Profit Strategy: 市場Evidenceなしに実行しない時
- Knowledge Compression: Raw情報を再利用Ruleへ変換する時

## Human Console Output

Human ConsoleにはRoute全文を出さない。

出すもの:

- Current Highest Trust Risk
- Next Rule Fix
- Rule Compliance
- Evidence Status
- Next Human Action 1つ

## AI Control Center Output

AI Control Centerには実行に必要な詳細を置く。

- Current Task
- Official Docs URLs
- Current Screen
- Required Inputs
- Prohibited Inputs
- Expected Output
- Completion Check
- Rule Checklist
- Evidence Requirement
