# Recovery Codes

## 現在状態

Recovery code inventory is not complete.

## 目的

Recovery codes prevent account loss when a phone, passkey, authenticator, or login method is unavailable.

## 今やること

For each critical service, generate / download / save recovery codes in Password Manager.

## 注意事項

- Never store recovery codes in Git.
- Never paste recovery codes into ChatGPT / Codex.
- Do not put screenshots in shared folders.

## Recovery

Keep at least one recoverable path for Google, GitHub, Vercel, Supabase, Cloudflare, and AI API consoles.
