# AI Control Center

Purpose: AI / Codex / Browser Agentが見る作業面の候補。

This is not a full implementation. Human Console remains the default entry for Human. AI Control Center exists to prevent AI-only details from making Human Console too long.

## Put Here

- Current Task
- Official Docs URLs
- Current Screen
- Required Inputs
- Prohibited Inputs
- Expected Output
- Completion Check
- Rule Checklist
- Evidence Requirement

## Do Not Put In Human Console

- Long docs
- Full internal task text
- Raw evidence
- Secret values
- AI-only implementation notes

## Current Status

Minimal candidate only. Use `index.html` as a static placeholder until Task Queue / Validation / Worker routing exists.
